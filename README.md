# Piano Keyboard

A simple tool to allow you to print a pattern of keys on a piano keyboard.

#### Intent

As musicians,
We need the ability to reference scales/ modes that are fun to modulate between,
That are hard to memorize without a good visualization method,
That take too much time to generate manually whereas we need to make that fast and easy
So that rapid experimentation with modulations can take place.


#### Vision

Allow you to generate charts like this one: https://www.etsy.com/listing/1012451620/piano-escale-major-modes-learn-piano
You should be able to pick the scales easily by clicking the keys, and then click "add" to add it to the chart batch that you'll be able to print off.


#### Capabilities

- Allow creation of printable scales so that 7 scales can be printed on one page and they look good.
- Build infrastructure that supports the Major modes
- Build infrastructure that can support Pentatonic modes
- Build infrastructure that can unknown modes
- Build support for multi-octave scales
- Build support for scales that go up/ down using different notes
